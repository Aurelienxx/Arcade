# Graphique.java

 Description de la classe Graphique:

Graphique est une classe Java qui sert à gérer la fenêtre d'affichage graphique, les éléments affichés et leurs interactions. Le constructeur Graphique crée une nouvelle fenêtre et initialise les dimensions TAILLE_X et TAILLE_Y. La méthode ajouter permet d'ajouter des objets à la fenêtre. Les méthodes selection et lectureMusiqueFond sont implémentées pour gérer les interactions de l'utilisateur et gérer la lecture de la musique dans le fond.

La classe Graphique utilise d'autres classes implémentées dans ce code, notamment Fenetre pour gérer la fenêtre graphique et ses interactions, Bouton pour représenter un bouton, Texture pour représenter une image, et Texte pour représenter du texte. La classe Graphique fonctionne en étroit étroit lien avec ces classes pour créer la fenêtre et gérer l'affichage des différents éléments.
