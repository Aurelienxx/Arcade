Code Java analysé : BoiteDescription.java

La classe BoiteDescription sert à gérer et afficher un ensemble de textes dans un cadre réglable, ainsi qu'une série d'icônes ou boutons. Elle utilise également des classes internes pour gérer les textes, boutons, joystick et affichage de hauts-scores. Elle peut charger le contenu d'un fichier externe texte et gère également l'affichage et le fonctionnement des boutons. Elle peut lire un fichier d'ajouts de bouton, d'un fichier de description et d'un fichier de hauts-scores.
