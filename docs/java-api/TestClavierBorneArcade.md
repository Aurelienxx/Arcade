# TestClavierBorneArcade.java

 Classe TestClavierBorneArcade.java

Description : Classe permettant de tester un clavier pour une borne d'arcade, simulant les touches et joysticks associés.

Responsabilités :
- Représenter les actions des joueurs sur les joysticks et boutons d'une borne d'arcade.
- Créer une instance de Fenêtre et l'ajouter à une application.

Attributs importants :
- fenêtre : instance de la fenêtre.

Méthodes publiques :
- main(String[]) : exécuter le programme en créant et affichant une fenêtre.

Relations :
- Interagit avec la classe Fenêtre pour afficher une fenêtre et les éléments graphiques associés.
